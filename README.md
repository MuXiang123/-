### 项目技术

使用Maven项目结构。数据库采用MySQL ，并用navicat可视化工具进行管理。数据传输交互采用servlet、ajax、json。前端UI采用layui框架，套用layuimini模板，使用基于jquery及javascript的jsp页面实现。使用IDEA作为开发工具，并将网页部署在Tomcat上运行。

- MySQL 8.0
- Tomcat 8.5以上（包括8.5）
- JDK 1.8

### 前端页面
